# **App Name**: ResumeCraft AI

## Core Features:

- User Authentication: Secure user authentication using email/password and Google sign-in.
- Drag-and-Drop Editor: Intuitive drag-and-drop resume editor with customizable sections.
- Resume Templates: Multiple professional resume templates (modern, classic, creative) to choose from.
- AI Content Suggestions: AI-powered tool to suggest content improvements for various sections of the resume.
- ATS Compatibility Check: Analyzes the resume for compatibility with Applicant Tracking Systems.
- PDF Export: Generates a PDF version of the resume.
- Cloud Storage: Secure cloud storage for user resumes.

## Style Guidelines:

- Primary color: A serene blue (#5DADE2) to inspire trust and professionalism.
- Background color: Light grey (#F0F4F8) for a clean and modern aesthetic.
- Accent color: A deep blue (#3498DB) to add subtle contrast and call attention to important elements.
- Body text: 'Inter' sans-serif font for a clean and readable experience.
- Headline text: 'Space Grotesk' sans-serif font for a modern and techy feel.
- Use clean, Material Design icons throughout the interface.
- Employ Material Design principles to ensure consistency and a user-friendly experience.
- Incorporate subtle animations to enhance the interactivity of the application, like section transitions or hover effects on interactive elements.
